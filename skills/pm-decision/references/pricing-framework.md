# 定价策略框架

定价是决策的一种。本文件为 pm-decision 的定价决策提供框架支持。

## 4 种定价模型

| 模型 | 适用场景 | 优势 | 风险 |
| --- | --- | --- | --- |
| 价值定价（Value-Based） | 差异化产品、ROI 可量化 | 捕获最大价值 | 需要深度用户理解 |
| 成本加成（Cost-Plus） | 成本敏感市场、内部产品 | 确保利润 | 忽略用户支付意愿 |
| 竞争定价（Competitive） | 饱和市场、同质化严重 | 市场对齐 | 价格战 |
| 免费增值（Freemium） | 网络效应、病毒式增长 | 低门槛、大用户量 | 高支持成本、转化率 2-5% |

**推荐**：先用价值定价确定区间，再用竞争定价验证。

## 定价心理学 6 法则

| 法则 | 应用 | 示例 |
| --- | --- | --- |
| 锚定效应 | 先展示高价选项 | Enterprise $99 → Pro $29 感觉便宜 |
| 损失厌恶 | 强调不使用的损失 | "每天 $0.97" > "每月 $29" |
| 装饰者效应 | 加一个不合理的选项 | 三选项：A $10 / B $29 / C $99 → B 看起来合理 |
| 精准定价 | 非整数感觉更精确 | $29.97 > $30 |
| 分层定价 | 3-4 层，推荐中间层 | Free / Standard / Pro / Enterprise |
| 承诺一致性 | 从免费开始，逐步升级 | 试用期 → 付费 → 高级 |

## 定价决策 Checklist

1. [ ] 明确定价目标（利润/增长/渗透/平价）
2. [ ] 量化用户价值（节省时间/增加收入/降低成本）
3. [ ] 分析成本结构（固定+变动，确保毛利率目标）
4. [ ] 研究竞品定价（参考非锚点）
5. [ ] 验证单位经济（LTV:CAC > 3:1，回本期 < 12 月）
