# 交付前检查

- [ ] 竞品范围明确（3-8 个，含类型分类）
- [ ] 分析维度 ≥ 3 个
- [ ] 对比矩阵完整（竞品 × 维度）
- [ ] 每个结论有来源引用
- [ ] 数据收集日期已标注
- [ ] 每个竞品有 ≥ 1 个优势和 ≥ 1 个劣势
- [ ] 置信度标注合理（不可全"高"）
- [ ] 差异化建议已列出
- [ ] **差异化建议中的技术路径已验证**（非 Magic Step）
- [ ] **引用的竞品功能有数据源支撑**（不是假设存在）
- [ ] 所有定量断言已标注来源（无来源的标注 [待验证]）
- [ ] 推断和建议已标注所依据的事实
- [ ] 核心结论有 >=2 个独立来源，或已标注 [单一来源-置信度低]
