# 商业模式模板

商业模式是策略讨论的一个主题类型。本文件为 pm-strategy-session 提供商业模式分析框架。

## 商业模式画布 9 要素

| 模块 | 核心问题 | PM 关注点 |
| --- | --- | --- |
| 用户群 | 服务谁？ | 具体到可验证的人群描述 |
| 价值主张 | 解决什么问题？ | 用户痛点 + 差异化方案 |
| 渠道 | 怎么触达？ | 获客路径 + 转化漏斗 |
| 用户关系 | 如何维护？ | 自助/人工/社区/共创 |
| 收入流 | 怎么赚钱？ | 收费模式 + 定价策略 |
| 核心资源 | 需要什么？ | 技术/数据/品牌/人才 |
| 关键活动 | 做什么？ | 产品/运营/增长 |
| 关键伙伴 | 谁帮忙？ | 供应商/渠道/战略联盟 |
| 成本结构 | 花多少钱？ | 固定/变动 + 单位经济 |

## 画布选择

| 框架 | 适用场景 |
| --- | --- |
| **Business Model Canvas** | 成熟业务、多用户群、复杂组织 |
| **Lean Canvas** | 早期创业、单用户群、需要快速验证 |

**PM 场景经验**：多数产品用 Lean Canvas 就够了。Business Model Canvas 更适合有多个业务线的公司。

## PM 场景常见收入模型

| 收入模型 | 适用产品类型 | 特点 |
| --- | --- | --- |
| SaaS 订阅 | 工具类、平台类 | 可预测、需高留存 |
| 按量计费 | API、云服务 | 公平但不可预测 |
| 交易抽成 | 平台、市场 | 依赖规模 |
| 广告/免费+增值 | 内容、社交 | 需要流量，转化率 2-5% |
| 一次性购买 | 软件、硬件 | 简单但无经常性收入 |
