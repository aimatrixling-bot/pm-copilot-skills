# 策略讨论执行指南

> 30 分钟结构化策略讨论的方法论和技巧。

---

## 1. 议题类型与处理策略

### 方向决策

| 维度 | 处理方式 |
|------|----------|
| 核心问题 | "我们该往哪走？" |
| 数据来源 | 竞品分析 + 市场趋势 |
| 评估标准 | 战略契合度 × 可行性 × ROI |
| 典型时长 | Phase 2 最长（需竞品数据） |

### 优先级决策

| 维度 | 处理方式 |
|------|----------|
| 核心问题 | "先做什么？" |
| 数据来源 | 产品指标 + RICE 评分 |
| 评估标准 | 价值 × 紧急度 × 依赖关系 |
| 典型时长 | Phase 3 最长（需方案对比） |

### 资源分配

| 维度 | 处理方式 |
|------|----------|
| 核心问题 | "人力/预算怎么分？" |
| 数据来源 | ROI 指标 + 团队容量 |
| 评估标准 | 投入产出比 × 风险 |
| 典型时长 | Phase 2-3 均衡 |

### 时间决策

| 维度 | 处理方式 |
|------|----------|
| 核心问题 | "什么时候做？" |
| 数据来源 | 市场窗口 + 团队容量 |
| 评估标准 | 紧急度 × 准备度 × 机会成本 |
| 典型时长 | Phase 2 最长（需市场数据） |

## 2. 时间管理技巧

### 30 分钟分配建议

```
Phase 1: 议题定义    ≤ 3 分钟  (10%)
Phase 2: 数据收集    ≤ 10 分钟 (33%)
Phase 3: 方案生成    ≤ 10 分钟 (33%)
Phase 4: 策略产出    ≤ 7 分钟  (24%)
```

### 时间告警机制

| 剩余时间 | 动作 |
|----------|------|
| Phase 1 超过 3 分钟 | 追问核心议题，打断发散 |
| Phase 2 超过 10 分钟 | 标注 [待补充]，不阻塞 |
| Phase 3 超过 10 分钟 | 强制收束到 Top-2 方案 |
| Phase 4 超过 7 分钟 | 记录当前决策状态，转为异步 |

## 4. 常见陷阱

| 陷阱 | 表现 | 应对 |
|------|------|------|
| 议题漂移 | 从 A 讨论到 B 到 C | 每次发言前确认"这回答了议题吗？" |
| 数据不足强行决策 | "就先这样吧" | 标注 [待补充]，设复盘日期 |
| 只有拥护没有反对 | 所有人同意但无人思考风险 | 要求"谁来当魔鬼代言人？" |
| 方案同质化 | 两个方案本质相同 | 要求一个方案必须与另一个有明显差异 |
| 决策拖延 | "再想想"、"下次再定" | 设定决策截止时间 |
