# PM Copilot Skills

29 structured PM Skills for Claude Code, covering the full product management lifecycle from problem framing to retrospective.

## Quick Install

```bash
# Install globally (available in all projects)
npx pm-copilot-skills

# Or install to current project only
npx pm-copilot-skills project
```

## What's Included

| Category | Skills |
|----------|--------|
| Discovery & Definition | pm-problem-frame, pm-discovery, pm-persona, pm-comp |
| Planning & Decisions | pm-prd, pm-rice, pm-decision, pm-roadmap, pm-backlog |
| Design & Prototype | pm-tech-spec, pm-wireframe, pm-prototype |
| Validation & Quality | pm-experiment, pm-metrics, pm-testing, pm-critique |
| Delivery & Communication | pm-launch, pm-eng-request, pm-solution-brief, pm-sync |
| Workflow & Growth | pm-feature-cycle, pm-writer-pipeline, pm-strategy-session, pm-data-analysis, pm-ost, pm-job-search, pm-urgent, pm-ai-patterns, pm-retro |

Plus shared references: quality gates, methodology KB (23 classic PM books), design patterns.

## Usage

After installation, use any skill in Claude Code:

```
/pm-prd Describe your product idea...
/pm-comp Compare ProductA vs ProductB...
/pm-critique Review this PRD...
```
